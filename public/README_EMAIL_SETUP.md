# Инструкция по настройке отправки писем через Mail.ru

## Настройка учетных данных

1. Откройте файл `public/send_email.php`

2. Найдите и замените следующие переменные на свои данные:

```php
$smtp_username = 'your_email@mail.ru'; // Ваш email на mail.ru
$smtp_password = 'your_password'; // Пароль приложения Mail.ru
$smtp_from_email = 'your_email@mail.ru'; // Ваш email (от кого будет приходить письмо)
```

3. **Важно**: Для Mail.ru нужно использовать пароль приложения, а не обычный пароль аккаунта.

## Как получить пароль приложения Mail.ru:

1. Зайдите на https://mail.ru/
2. Войдите в свой аккаунт
3. Перейдите в Настройки → Безопасность
4. Найдите раздел "Пароли приложений"
5. Создайте новый пароль приложения для почтового клиента
6. Скопируйте сгенерированный пароль и используйте его в `$smtp_password`

## Настройка получателя

В файле `send_email.php` найдите строку:
```php
$to_email = 'crystalluxe@mail.ru'; // Email из контактов
```

Измените на email, куда должны приходить заявки.

## Проверка работы

1. Убедитесь, что на сервере установлен PHP 7.4 или выше
2. Проверьте, что PHPMailer находится в папке `public/PHPMailer/`
3. Откройте форму "Записаться на консультацию" на сайте
4. Заполните и отправьте форму
5. Проверьте почтовый ящик получателя

## Устранение проблем

- Если письма не отправляются, проверьте логи ошибок PHP
- Убедитесь, что на сервере разрешены исходящие SMTP соединения
- Проверьте правильность пароля приложения
- Убедитесь, что используется правильный SMTP-сервер (smtp.mail.ru, порт 465, SSL)
